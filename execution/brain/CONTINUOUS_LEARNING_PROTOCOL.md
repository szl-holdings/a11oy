# Continuous learning protocol

Status: MODELED. This protocol does not claim autonomous weight training.

1. Observe a source event with actor, time, license, scope, and immutable revision.
2. Validate and admit it as memory or quarantine it.
3. Retrieve relevant prior evidence and expose contradictions.
4. Produce a bounded improvement proposal with a stated hypothesis.
5. Evaluate in a sandbox against locked tests and safety gates.
6. Require an authorized human for repository changes, deployment, or training use.
7. Record the decision, deployment revision, measured outcome, and rollback path.
8. Promote the outcome to reusable memory only after verification.

Learning means evidence-backed changes to memory, retrieval, policy, prompts, code, or
models through this governed loop. Model-weight changes require a separate dataset,
training configuration, evaluation, artifact digest, and signed training receipt.
