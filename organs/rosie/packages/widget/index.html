rosie-widget

Ambient operator widget. The floating launcher sits bottom-right. Click it, or press ⌘K / Ctrl-K anywhere, to expand the panel. Press Esc to collapse.

  • Floating button bottom-right, accent per host app.
  • Click → panel slides in from the right, focus trapped inside.
  • ⌘K / Ctrl-K → command palette in the panel.
  • Esc → panel collapses, focus returns to the button.
  • With no live rosie-api the widget shows an offline message and makes no network calls.
