## Founder Action Checklist (FA-001)

```bash
# 1. Download tarball from HF
python3 -c "
from huggingface_hub import hf_hub_download
path = hf_hub_download(repo_id='SZLHOLDINGS/sentra-source',
                       filename='sentra-uds-0.3.0.tar.zst', repo_type='dataset')
print(path)
"

# 2. Verify sha256
sha256sum sentra-uds-0.3.0.tar.zst
# Expected: 659a2fb6aa1ea4977c43dd02d485e42369c8b51520d81df5d220f5c58421c2db

# 3. Sign
cosign sign-blob --key <org-dev-private-key.pem> \
  sentra-uds-0.3.1.tar.zst \
  --output-signature sentra-uds-0.3.1.tar.zst.sig

# 4. Push container
docker push ghcr.io/szl-holdings/sentra:uds-v0.3.1

# 5. Upload to release
gh release upload uds-v0.3.1 --repo szl-holdings/sentra \
  sentra-uds-0.3.1.tar.zst \
  sentra-uds-0.3.1.tar.zst.sig \
  sentra-uds-0.3.1.tar.zst.sha256 \
  sentra-uds-dev.pub
```
