## Determinism: tail_hash Fix

The critical determinism bug in v0.5 development:

**Problem:** `RAGReceipt.artifact_path` stored tempfile paths (`/tmp/tmpXXXXX.json`).  
These differed between process runs → `tail_hash` was non-deterministic.

**Fix:** Normalize `artifact_path=""` when computing `tail_hash` in `ouroboros_infinity.py`:

```python
# In CycleTail hash computation:
normalized = receipt.model_copy(update={"artifact_path": ""})
tail_data = canonical_json(normalized.model_dump())
```

After this fix: 5× replay of `agi_brain.py` is byte-identical.
