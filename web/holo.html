WHAT THIS PROVES

The shared kit /static/shared/szl_holo3d.js renders a defense-grade holographic shell with a proof-DAG graph, a Λ-driven trust sphere, and a signed-pulse edge animation. All data shown is SAMPLE (illustrative fixtures, not a live feed). Λ is Conjecture 1 — a bounded score < 1.0, not a theorem. Trust is never 100%.
Build references (no runtime calls): three.js / React-Three-Fiber (pmndrs/react-three-fiber), deck.gl (visgl/deck.gl), CesiumJS (CesiumGS/cesium). The globe shown is a procedural SAMPLE wireframe, not a CesiumJS tile basemap (a 0-CDN local-tile basemap is ROADMAP).
